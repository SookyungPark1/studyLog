## 📌 목적(Why)
- 배운 내용을 **짧게 기록**하고, 다음에 **빠르게 재사용**하기 위해
- 같은 문제를 반복해서 헤매지 않도록 **나만의 레퍼런스** 만들기

- 코드는 **최소 실행 예시(MVP)** 위주로
